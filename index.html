Local First

Memory, journals, configs, and prototypes live in files the owner can inspect directly.

Hybrid Reasoning

Codex handles architecture and integration. DeepSeek helps with bounded bulk work. Local tools judge behavior.

Small Tools

Operating Loop

  1. Frame the work clearly.
  2. Delegate bulk thinking when useful.
  3. Implement locally and verify directly.
  4. Write down what changed.
